## sqliteDBPractice

# Install Golang 
- https://go.dev/doc/install

# Clone directory in GOPath

# Run commands:
- go mod init 
- go mod tidy 
- go build

# Start server: 

- go run main.go 

# Usage

- Query database using query parameters 

- Ex. /payments?date=<1642248000>&status=<failed/success>

- Without any query parameters returns entire payments table
